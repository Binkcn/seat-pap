# seat-pap

A module for [SeAT](https://github.com/eveseat/seat) that tracks Pilot Attendance Points (PAP) for fleets and campaigns.

[![Latest Stable Version](https://img.shields.io/packagist/v/binkcn/seat-pap.svg?style=flat-square)]()
[![License](https://img.shields.io/badge/license-GPLv2-blue.svg?style=flat-square)](https://raw.githubusercontent.com/eveseat-plugins/seat-pap/master/LICENSE)

This plugin allows corporations and alliances to track pilot attendance in fleets and campaigns through a point-based system (PAP). Authorized Fleet Commanders (FCs) can create fleets, record participants, and grant PAP points.

## Features

- **PAP Tracking**: Record and manage attendance points for pilots
- **Fleet Management**: Authorized FCs can create and manage fleets
- **Point Distribution**: Assign PAP points to fleet participants
- **Access Control**: Granular permissions for FCs and administrators

## Quick Installation:

Please see the SeAT docs for installation instructions [HERE](https://eveseat.github.io/docs/community_packages/).

The composer string to use is `binkcn/seat-pap`

After installation and configuration, you should see a 'Pilot Attendance Program' or 'PAP' link in your SeAT navigation menu.
